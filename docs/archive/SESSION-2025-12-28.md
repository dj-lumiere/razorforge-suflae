## Key Language Rules Established

1. **`None` not `none`** - capitalized
2. **Suflae types**: `Integer`, `Decimal`, `Fraction`, `bool`, `Text` (not `Float`, `Bool`)
3. **Fixed-width types**: `s32`, `f32`, `f64`, etc. (available in both languages)
4. **Records**: No `var` keyword for fields - just `fieldName: Type`
5. **Choice/Variant cases**: Any casing allowed (not forced SCREAMING_SNAKE_CASE)
6. **Access block downgrade**: `seizing`→`hijacking`/`viewing`, `inspecting`→`viewing`
7. **No inline timeout** in access blocks - use `try_seize(timeout: X)` method instead
8. **Crashable protocol**: Uses `message()` method (not `crash_message()`)
